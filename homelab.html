🧪 My Homelab

Exploring self-hosted tech, security testing, and server experiments

Overview

My homelab is a personal space for experimenting with networking, virtualization, and cybersecurity tools. It includes self-hosted services, monitoring, and secure networking setups.

Technologies Used

  • Proxmox VE for virtualization
  • Pi-hole DNS sinkhole
  • Docker & Portainer for container management
  • WireGuard VPN
  • OpenMediaVault NAS

Current Projects

  • 🔐 Self-hosted password manager behind a reverse proxy
  • 📊 Homeassistant
  • 🧼 Reverse DNS to have access my remote server

More to Come

As I expand the lab, I plan to document setup guides, diagrams, and logs for reference and sharing. Stay tuned!
